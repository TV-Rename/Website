{:.cli}
/scan

Tell TV&nbsp;Rename to run a full scan.

{:.cli}
/recentscan

{:.cli}
/quickscan

Tell TV&nbsp;Rename to run a quick scan.

{:.cli}
/doall

Tell TV&nbsp;Rename execute all the actions it can (Need to be specify which scan type is required - from above 3 options).
